### NBB-706 - Drop suffix drift, convert stateless classes, remove barrels, delete shims, update final docs

**Goal:** Make the new structure the current structure, not a compatibility layer over the old one.

**Scope:**
- Drop `Service`/`Executor` suffix drift where the path supplies context.
- Convert verified stateless classes to modules/functions.
- Remove accidental `__init__.py` barrel re-exports.
- Delete temporary shims and forwarding-only modules unless a documented compatibility boundary remains.
- Update final architecture docs to describe the new truth.

**Acceptance criteria:**
- No forwarding-only modules or temporary shims remain without documented compatibility reason.
- Old mechanism taxonomy is gone from current guidance.
- New names use local context instead of ancestry repetition.

**Verification:** Search for suffix drift, re-export barrels, and legacy imports.

**Traceability:** Covers audit Phase 3a/3c/3e and final cleanup.
